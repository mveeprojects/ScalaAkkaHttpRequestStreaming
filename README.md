# Scala Akka Http Request Streaming

Hello actor route:
`curl -X PUT http://localhost:8080/hello/howareyou`

Add item route: `curl -X PUT http://localhost:8080/item/abc`

Delete item route: `curl -X DELETE http://localhost:8080/item/abc`
